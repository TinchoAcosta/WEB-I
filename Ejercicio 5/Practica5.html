Operadores Lógicos

Índice

  1. OR
  2. AND
  3. NOT

OR

El operador 'Or' o 'O' asocia dos o más partes de un problema, validando que almenos una de las partes sea cierta para poder continuar

AND

El operador 'And' o 'Y' asocia dos o más partes de un problema, validando que todas las partes sean ciertas para poder continuar

NOT

El operador 'Not' o 'No' asocia una parte de un problema, validando que sea falsa para poder continuar
